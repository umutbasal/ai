# AI Skills

A collection of AI skills

## Tool Skills

- **Semgrep Static Analysis Tool Skill**  
  Static analysis and security scanning using [Semgrep](https://semgrep.dev). Find bugs, security vulnerabilities, and enforce code standards with pattern matching.

- **Comby Structural Code Search and Rewrite Tool Skill**  
  Structural code search and rewriting using [Comby](https://comby.dev). Syntax-aware refactoring that understands code structure.

## Testing

Each skill includes comprehensive tests to ensure functionality:

```bash
# Test Semgrep skill
cd tests/semgrep
bun test

# Test Comby skill
cd tests/comby
bun test
```
